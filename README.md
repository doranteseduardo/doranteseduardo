### Research

**[Reverse-Engineering the INE Credential QR Codes](https://github.com/doranteseduardo/ine-qr-re)**
— Independent security analysis of the cryptographic system embedded in Mexican voter
credentials. Reverse-engineered a seven-layer pipeline (AES-CBC-256, chained 8,192-bit RSA,
SIMD/NEON key derivation, ECDSA) in a native ARM64 library. Implemented a functional decoder
via Unicorn Engine emulation. *March 2026.*
