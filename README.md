### Research

**[Reverse-Engineering the INE Credential QR Codes](https://github.com/doranteseduardo/ine-qr-re)**
— Independent security analysis of the cryptographic system embedded in Mexican voter credentials. Reverse-engineered a seven-layer pipeline (AES-CBC-256, chained 8,192-bit RSA, SIMD/NEON key derivation, ECDSA) in a native ARM64 library. Implemented a functional decoder via Unicorn Engine emulation. _March 2026._

**[SAT FIEL PKI Cryptanalysis](https://doranteseduardo.com/SAT%20FIEL%20PKI%20Security%20Analysis.pdf)**
— Structural and cryptographic analysis of Mexico's e.firma digital identity infrastructure, used by 27 million taxpayers. Documented deprecated ciphers, weak key derivation, and short salts. Includes a proof-of-concept attack pipeline with GPU benchmarks.
